# 🌤 Weather App

A Python weather application that displays:

- Current weather
- 5-day forecast
- Temperature
- Humidity
- Wind Speed
- Pressure
- Sunrise & Sunset

## Technologies

- Python
- Requests
- OpenWeatherMap API

## Installation

```bash
pip install -r requirements.txt


Run:

```bash
python main.py
